About Me

I'm a NYC-based Ruby and JavaScript web developer who loves coding. My journey into computers began in the 8th grade, when I built my first computer out of parts bought from newegg and ebay. After taking Harvard's Intro to CS MOOC (taught by an absolute legend) I decided to grok code more seriously and start building my own projects while partaking in an immersive full-stack web developer course.
